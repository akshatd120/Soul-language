# Soul Programming Language Compiler

**Description:**

This repository contains a compiler for the Soul programming language, which is currently a hobby project in development. The compiler is written in C++ and is actively working towards becoming self-hosted.

**About Soul Programming Language:**

Soul is a hobby programming language that is still a work in progress. It is designed to be expressive and versatile.

**Compiler Details:**

- Language: C++
- Self-hosting: In progress

Please note that this project is under active development, and contributions are welcome.
